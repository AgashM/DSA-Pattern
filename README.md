# 🚀 DSA Pattern Map

A structured repository to master **Data Structures & Algorithms** using **pattern-based problem solving** instead of random practice.

---

## 📌 Goal

* Identify patterns quickly
* Optimize brute force solutions
* Build strong problem-solving intuition for coding interviews

---

## 🧠 Patterns Covered

### Arrays & Strings

* Two Pointers
* Sliding Window
* Prefix Sum
* Hashing
* Merge Intervals
* Sorting

### 🔜 Coming Next

* Linked List
* Stacks & Queues
* Trees
* Graphs
* Dynamic Programming

---

## 📂 What Each Problem Includes

* ✅ Optimized solution
* 🧩 Pattern explanation
* ⚡ Time & Space Complexity

---

## 🔥 Approach

For every problem:

1. Identify the pattern
2. Start with brute force
3. Optimize using the right approach
4. Document key learnings
